Documentation for Boring Math PyPI projects

The Boring Math project consists of Python PyPI mathematical packages. The purpose of this collection projects is to host and provide infrastructure for my Python mathematical hobby projects. The Boring Math library consists of Python PyPI namespace packages all of which are under the bm. "namespace."

The project name was suggested by my then 13 year old daughter Mary.
